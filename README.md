# Swingy

Swingy is a simple Java-based role-playing game project that demonstrates core game development concepts such as character creation, movement, combat, and progression.

## Features
- Create and customize a player character
- Explore a game map
- Engage in turn-based combat
- Manage inventory and equipment
- Save and load game progress

## Requirements
- Java 17 or newer
- A Java IDE such as IntelliJ IDEA or VS Code, or a build tool like Maven

## Getting Started
1. Clone the repository.
2. Open the project in your preferred Java IDE.
3. Build and run the main application class.

```bash
    mvn compile
    mvn package
    java -cp target/my-swing-app-1.0-SNAPSHOT.jar com.example.App

```
